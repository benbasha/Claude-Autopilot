Under the Hood

Mobile Control Architecture

  • Local Web Server - Spins up automatically with Claude Autopilot
  • Real-time Queue Sync - Mobile interface stays in sync with VS Code
  • Responsive Design - Works on any phone, tablet, or device
  • Security First - Local network only, never leaves your machine

Feature Requirements

  • Claude Autopilot: Extension v0.1.2+ required
  • Devices: Any phone, tablet, or computer with browser
  • Network: Same local network as your VS Code machine
  • Setup: Zero configuration - works automatically
